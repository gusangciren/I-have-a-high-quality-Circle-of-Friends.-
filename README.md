这是一本被“标题”所耽误的书籍，标题从里到外都透露着一个字——俗，俗不可耐。
我们的家庭教育还是学校教育，都让我们觉得谈钱就是俗。
他们都喜欢它，但不敢大声说出来我喜欢它，更不敢明面追求它。
因此，很多人不敢买，买了也是偷偷看。

看的都明白，李笑来书里每一个“钱”的字样，一个也没有。
实践的都明白，李笑来处处在说赚钱。只是李笑来就像一个害羞的男孩，不敢直接说，我喜欢你，而是说我们好像在那里见过。

李笑来讲了赚钱的底层原理。比如注意力＞时间＞金钱。
